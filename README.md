# LuontoPeli

Koulutyönä tehty luontopeli.  
*Mobiilisofta jossa käyttäjä voi tunnistaa kuvista kasveja sieni ja eläimiä.*
*Sovelluksessa voit aloittaa kävely session, kartta seuraa askeleita ja sijainteja.*
*Otettuasi kuvan voit jättää löytöön kommentin joka näkyy markkerina kartalla*
